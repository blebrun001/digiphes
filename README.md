# IPHES-CERCA Website

Minimal one-page static website for the IPHES-CERCA project.

## Current Scope
- Minimal header with white logo
- Black-and-white framed layout ("cartouche" style)
- Main sections: hero, key figures, visual roadmap
- Responsive layout for small and large screens

## Tech
- `index.html`
- `assets/css/styles.css`
- `assets/js/i18n.js`
- `assets/js/app.js`

## Run Locally
```bash
python3 -m http.server 8080
```

Open `http://localhost:8080`.
